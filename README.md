# Sidescrolling Autotile Tileset Template

16px tileset template for sidescrolling games with connected textures. 

Also included is a TileSet resource with terrain and collision set up so you can hopefully get started really fast.

![here it is](https://github.com/MemeKing/Autotile-Tileset-Template-16px/blob/main/addons/tileset_template/16x-tileset-template.png)
